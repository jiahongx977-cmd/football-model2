部署说明｜JAX的AI模型 V7.2 UI视觉统一优化版

升级来源：
从 V7.1 继续改，不是从旧版本改。

上传方式：
1. 解压 zip。
2. 打开 GitHub 仓库：
   https://github.com/jiahongx977-cmd/football-model2
3. 点击 Add file → Upload files。
4. 上传解压后的全部文件，不要上传 zip 压缩包本身。
5. 点击 Commit changes。
6. 等待 Vercel 自动部署。

环境变量：
Vercel 项目中继续使用：
THE_ODDS_API_KEY

注意：
不要把真实 API Key 写入网页文件。
本工具只做体育数据分析、概率模型学习、Dutching 数学测算与赛后复盘训练，不提供投注建议或收益承诺。
