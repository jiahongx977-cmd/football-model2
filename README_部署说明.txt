JAX的AI模型 V7.2 UI视觉统一优化版

本版本从 V7.1 Dutching流程强化 + 触发来源精简版继续升级。

本次升级重点：
1. 首页快捷入口优化
   - 删除“今日足球”快捷按钮。
   - 剩余快捷入口尽量排成一排。
   - “世界杯比赛”按钮由紫色改成橙色，与世界杯冠军赔率按钮风格更统一。

2. 主按钮视觉统一
   - “获取近期比赛”改成红色大按钮。
   - 全站红色主操作按钮统一尺寸、圆角、字体和颜色。
   - 红色按钮用于提醒用户这是主要操作步骤。

3. Dutching 输入区视觉调整
   - 预算输入不再做大面积高亮。
   - 预算输入框只保留橙色描边。
   - 分配方式选择框也只保留橙色描边。
   - 保持界面干净，避免整块页面过度发橙色。

4. 保留 V7.1 功能
   - 波胆推荐表删除重复说明列。
   - 触发来源动态精简。
   - 本场波胆快捷区按钮位于表格下方。
   - Dutching三种模式保留。

项目定位：
这是一个体育数据分析、概率模型学习、赔率结构研究、Dutching 数学分配、赛后复盘训练工具。
不是投注建议工具，不提供带单、跟单、资金托管、代买服务或收益承诺。
页面中出现的赔率、ROI、Dutching、回报等内容均为数学测算示例。
