1. 手机端 Dutching 表格横滑方式修正
   - 不再把表格做成卡片。
   - 不再让整页内容被横向撑开。
   - 表格保持电脑端表格样式。

   - 本场波胆 Dutching 快捷区表格。
   - Dutching 总页表格。
   - 本场波胆推荐表。

3. 保留功能
   - 手机端导航“Dutching投注”不变。
   - Dutching 逻辑、按钮、触发来源等功能不变。
