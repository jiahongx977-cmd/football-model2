JAX的AI模型 V6.8 动态比分推演修正版

本版本从 V6.7 复盘重复模块清理版继续升级。

本次升级重点：
1. 修复“本场波胆推荐”固定模板问题。
   - 不再机械套用 1-1、2-1、1-0、0-1、0-0、2-2。
   - 每场比赛会根据赔率结构、隐含概率、强弱关系、天气、教练、阵容、体能、裁判、盘口热度等因素动态推演。
   - 如果两场比赛模型判断接近，比分可以相似；但必须是模型计算相似，不是代码固定。

2. API赔率明细升级。
   - 分析结果中会显示主胜/平局/客胜平均赔率。
   - 显示主胜/平局/客胜隐含概率。
   - 显示大小球倾向和模型预估进球。
   - 若赔率资料不足，会明确提示“保守推演”。

3. 分析解释升级。
   - 不只输出波胆表。
   - 会说明为什么推这些比分，例如赔率方向、天气、教练、体能、裁判、盘口热度等因素如何影响。

4. Dutching 操作顺序优化。
   - 总预算输入位置保持不变。
   - 按钮顺序改为：添加一行 → AI计算权重 → 计算 Dutching + ROI。
   - 先计算权重，再计算建议金额、命中回报、净理论净值、ROI。

项目定位：
这是一个体育数据分析、概率模型学习、赔率结构研究、Dutching 数学分配、赛后复盘训练工具。
不是投注建议工具，不提供带单、跟单、资金托管、代买服务或收益承诺。
页面中出现的赔率、ROI、Dutching、回报等内容均为数学测算示例。

部署方法：
1. 解压本压缩包。
2. 上传全部文件到 GitHub 仓库 football-model2，覆盖原文件。
3. Vercel 会自动重新部署。
4. API Key 仍然只放在 Vercel 环境变量 THE_ODDS_API_KEY 中，不要写入 index.html。
