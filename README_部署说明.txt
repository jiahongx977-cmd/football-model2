JAX的AI模型 V5.0 AI波胆权重版

本版升级：
1. 删除/替换“重新同步比分池”按钮，改为“AI计算波胆权重”。
2. Dutching 快捷区新逻辑：
   - 用户自己填写想买的波胆
   - 填预算
   - 可选填赔率
   - 点击“AI计算波胆权重”，系统根据当前比赛模型给这些波胆重新打权重
   - 点击“计算本场波胆 Dutching”，系统计算建议金额、命中回报、净收益、ROI
3. AI权重依据：
   - 比分结构
   - 比赛性质
   - 强弱关系
   - 天气
   - 教练目的
   - 阵容状态
   - 赛程体能
   - 裁判因素
   - 可选赔率价值
4. Dutching 投资分配与模型推荐分开处理：
   - AI权重负责判断哪些比分更值得覆盖
   - Dutching负责按预算和赔率分配资金
