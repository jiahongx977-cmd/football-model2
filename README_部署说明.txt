JAX的AI模型 V7.0 波胆动态推演 + Dutching三模式高亮版

本版本从 V6.9 修正确认版继续升级。

本次升级重点：
1. 波胆推荐权重离散修正
   - 修复概率异常：主胜/平局/客胜不能出现负数。
   - 波胆权重不再过度平均。
   - 强弱悬殊局会拉高主线权重；均势局才相对分散。
   - 加入置信度逻辑：赔率越清晰、强弱越明显，主线越集中。

2. 剧本来源解释
   - 波胆推荐表新增“触发来源”列。
   - 每个比分会显示由哪些因素触发，例如赔率支持、热门过热、天气、体能、裁判、大小球、赛制等。
   - 剧本类型会根据比赛状态动态变化。

3. Dutching 三种模式
   - 等回报 Dutching（利润平摊型）
   - AI权重倾斜型
   - 阶梯利润最大化型

4. 波胆/冠军 Dutching 页面优化
   - 比分/结果、赔率、总预算仍由用户手动输入。
   - AI只负责计算权重，不自动改用户填的比分和赔率。
   - 分配方式选择器放在“添加一行”和“AI计算权重”之间。
   - 分配方式区域做高亮显示，提醒用户这里可以切换模式。

5. 本场波胆 Dutching 快捷区同步
   - 快捷区的分配方式也同步三种模式。
   - 等回报、AI权重倾斜、阶梯利润最大化三种逻辑保持一致。

6. 字段名称优化
   - “命中回报”更准确显示为“命中总返还”。
   - “净理论净值”更准确显示为“命中净收益”。
   - 页面补充公式说明：命中总返还 = 赔率 × 建议模拟金额；命中净收益 = 命中总返还 - 总预算。

项目定位：
这是一个体育数据分析、概率模型学习、赔率结构研究、Dutching 数学分配、赛后复盘训练工具。
不是投注建议工具，不提供带单、跟单、资金托管、代买服务或收益承诺。
页面中出现的赔率、ROI、Dutching、回报等内容均为数学测算示例。

部署方法：
1. 解压本压缩包。
2. 上传全部文件到 GitHub 仓库 football-model2，覆盖原文件。
3. Vercel 会自动重新部署。
4. API Key 仍然只放在 Vercel 环境变量 THE_ODDS_API_KEY 中，不要写入 index.html。
