JAX的AI模型 V7.6 手机端适配优化版

本版本从 V7.5 合规按钮清理版继续升级。

本次升级重点：
1. 手机端顶部导航优化
   - 解决手机上“波胆/冠军 Dutching”等按钮文字被挤压、截断的问题。
   - 手机端自动简化为“波胆/冠军”。
   - 导航按钮改成更适合手机点击的网格排列。

2. 手机端快捷入口优化
   - 快捷入口在手机端改为两列。
   - 按钮更高、更容易点击。

3. 手机端 Dutching 表格优化
   - 本场波胆 Dutching 快捷区表格在手机端改成卡片式布局。
   - 每一行都显示字段名：剧本、波胆、模型权重、赔率等。
   - 输入框不再被压成很窄的小框。
   - 主 Dutching 总页表格手机端也同步卡片化。

4. 手机端主按钮优化
   - 红色主按钮在手机端占满宽度。
   - 更适合抖音用户用手机打开操作。

项目定位：
这是一个体育数据分析、概率模型学习、赔率结构研究、Dutching 数学分配、赛后复盘训练工具。
不是投注建议工具，不提供带单、跟单、资金托管、代买服务或收益承诺。
页面中出现的赔率、ROI、Dutching、回报等内容均为数学测算示例。
