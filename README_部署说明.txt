JAX的AI模型 V6.6 复盘模块彻底修正版

本版重点修复：
1. 赛后复盘页会出现新的 V6.6 复盘历史模块。
2. 从“联网选比赛”点“保存到赛后复盘”后：
   - 只保留同场比赛最新一条“赛前待复盘”记录
   - 不会再连续出现一堆重复记录
3. 每条历史记录内可以直接补：
   - 实际比分
   - 模型判断结果
   - 误差类型
   - 赛后复盘备注 / 后续修正建议
4. “保存赛后结果”会把该条记录状态改为“已填写赛后结果”。
5. “保存当前复盘记录”强制生效。
6. “查看复盘历史”强制生效。
7. “清空复盘历史”会清空所有旧版本复盘历史 key，不会再清不掉。
8. 可单独删除某条复盘记录。
9. 可把某条复盘误差写入模型反馈。
