JAX的AI模型 V7.1 Dutching流程强化 + 触发来源精简版

本版本从 V7.0 波胆动态推演 + Dutching三模式高亮版继续升级。

本次升级重点：
1. 波胆推荐表展示优化
   - 删除重复的“说明”列。
   - 保留“触发来源”列。
   - 触发来源根据当前比赛赔率、强弱、天气、教练、阵容、体能、裁判、盘口热度和比分剧本动态生成。
   - 触发来源可以详细，但不能每场固定一样、不能每行复制同一堆内容。

2. 本场波胆 Dutching 快捷区流程优化
   - 本场波胆模拟预算输入框高亮。
   - 分配方式继续高亮。
   - 三个核心按钮移动到表格下方：
     AI计算波胆权重 → 计算本场波胆 Dutching → 同步到 Dutching 总页。
   - 表格下方按钮改为红色大按钮，符合“先填赔率，再计算”的操作顺序。

3. 波胆/冠军 Dutching 总页流程优化
   - 总预算输入框高亮。
   - 操作区改为纵向流程：
     添加一行 → 分配方式 → AI计算权重 → 计算 Dutching + ROI。
   - 比分/结果、赔率和预算仍由用户手动输入。
   - AI只负责计算权重和分配金额，不自动修改用户填写的比分与赔率。

项目定位：
这是一个体育数据分析、概率模型学习、赔率结构研究、Dutching 数学分配、赛后复盘训练工具。
不是投注建议工具，不提供带单、跟单、资金托管、代买服务或收益承诺。
页面中出现的赔率、ROI、Dutching、回报等内容均为数学测算示例。

部署方法：
1. 解压本压缩包。
2. 上传全部文件到 GitHub 仓库 football-model2，覆盖原文件。
3. Vercel 会自动重新部署。
4. API Key 仍然只放在 Vercel 环境变量 THE_ODDS_API_KEY 中，不要写入 index.html。
