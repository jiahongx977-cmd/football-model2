JAX的AI模型 V6.9 操作流程与盘口中文解释优化版

本版本从 V6.8 动态比分推演修正版继续升级。

本次升级重点：
1. 首页操作流程优化
   - “联网设置”放到最上面，作为第一步。
   - “近期比赛”放到联网设置下面。
   - 顶部增加操作流程提示：获取近期比赛 → 选择比赛 → 把API赔率带入模型 → 开始完整模型分析。

2. 近期比赛显示优化
   - 近期比赛区域固定高度显示，不再撑长整个页面。
   - 用户可以在近期比赛框内手动上下滑动。
   - 近期比赛英文队名增加中文翻译，例如 Mexico vs South Africa｜墨西哥 vs 南非。

3. 关键按钮优化
   - “把API赔率带入模型”放到“刷新赔率明细”前面。
   - “把API赔率带入模型”改成红色大按钮。
   - “开始完整模型分析”改成红色大按钮。
   - API带入按钮只负责把胜平负概率带入模型，不自动替用户完成最终分析，避免流程混乱。

4. API赔率明细增加中文解释
   - H2H = 胜平负 / 主胜 / 平局 / 客胜。
   - Totals = 大小球 / 全场总进球。
   - Over 2.5 = 大2.5，全场3球或以上。
   - Under 2.5 = 小2.5，全场0-2球。
   - Spreads = 让球盘。
   - -1.25 = 让1.25球。
   - +1.25 = 受让1.25球。

项目定位：
这是一个体育数据分析、概率模型学习、赔率结构研究、Dutching 数学分配、赛后复盘训练工具。
不是投注建议工具，不提供带单、跟单、资金托管、代买服务或收益承诺。
页面中出现的赔率、ROI、Dutching、回报等内容均为数学测算示例。

部署方法：
1. 解压本压缩包。
2. 上传全部文件到 GitHub 仓库 football-model2，覆盖原文件。
3. Vercel 会自动重新部署。
4. API Key 仍然只放在 Vercel 环境变量 THE_ODDS_API_KEY 中，不要写入 index.html。
